# Todo App (JavaScript)

A simple and interactive **Todo App** built using **HTML, CSS, and JavaScript**.  
It allows users to add tasks, mark them as completed, and delete them with smooth UI interactions.

---

## Features

-  Add new tasks
-  Mark tasks as done (with animation & icon)
-  Delete tasks
-  Prevent empty task submission
-  Clean and responsive UI

---

##  Tech Stack

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Font Awesome (icons)
